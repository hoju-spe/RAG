## Projects

### PICO - RAG 기반 AI 면접 답변 코치

사용자의 면접 질문과 답변을 입력받아 유사한 STAR 답변 사례를 검색하고, GPT 모델을 활용해 답변을 개선하는 RAG 기반 면접 코칭 서비스입니다.

- 위치: [`rag-interview-coach`](./rag-interview-coach)
- 주요 기술: Python, OpenAI API, ChromaDB, Gradio
- 주요 기능: 유사 답변 검색, STAR 답변 개선, 개선 전/후 평가, 답변 톤 분석
